## Why

Go Streaming Indonesia membutuhkan landing page one-page untuk meningkatkan konversi dari traffic organik (Google/IG/TikTok) menjadi inquiry WhatsApp. Saat ini tidak ada website yang menampilkan 8 layanan (Multicam/Livestream, Sound System, LED Videotron, Photobooth Classic/AI, 360 Booth, EO, Produksi Event) secara terstruktur dengan CTA jelas dan optimasi SEO lokal Balikpapan.

## What Changes

- Landing page one-page responsif dengan sticky navigation dan floating WhatsApp button
- 8 service cards dengan prefilled WhatsApp message per layanan
- Portfolio gallery dengan lightbox dan kategori tabs
- Contact form yang redirect ke WhatsApp dengan template message
- SEO optimization: local schema markup, meta tags, Core Web Vitals
- Mobile-first design dengan bottom quick navigation
- **Logo client section dengan animasi scroll otomatis**
- **Counter animasi untuk statistik (500+ project, 5+ tahun, 100% kepuasan)**
- **WhatsApp chat widget interaktif**
- **Section Tips & Artikel dengan popup modal detail**
- **Lazy loading untuk optimasi performa gambar**
- **Instagram icon integration di footer**
- **Gradient backgrounds dengan rounded containers untuk visual variety**
- **Mobile menu active state highlighting untuk better UX**
- **Dark theme sections (Paket & FAQ) untuk kontras visual**
- **Cloudflare Pages deployment dengan CDN optimization**
